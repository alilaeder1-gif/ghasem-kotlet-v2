# 🤖 ربات مدیریت گروه تلگرام

ربات مدیریت گروه با قابلیت چت هوشمند با هوش مصنوعی

## ✨ قابلیت‌ها

- **مدیریت کاربران**: بن، آبن بن، کیک، میوت، آنمیوت
- **خوشامدگویی**: پیام خودکار خوشامدگویی به اعضا جدید
- **قوانین گروه**: تنظیم و نمایش قوانین
- **تشخیص اسپم**: حذف خودکار پیام‌های اسپم
- **جلوگیری از Flood**: محدود کردن سرعت ارسال پیام
- **چت هوشمند**: پاسخ با هوش مصنوعی (HuggingFace)
- **پین پیام**: قابلیت پین کردن پیام‌ها

## 🚀 نصب

### 1. نصب پیش‌نیازها

```bash
pip install -r requirements.txt
```

### 2. تنظیم فایل .env

فایل `.env.example` رو به `.env` تغییر نام بدید و مقادیر زیر رو پر کنید:

```
BOT_TOKEN=توکن_بات
HUGGINGFACE_API_KEY=کلید_API
```

### 3. اجرای ربات

```bash
python bot.py
```

## 📋 دستورات

### دستورات ادمین

| دستور | توضیح |
|-------|-------|
| `/ban` | بن کاربر (روی پیام ریپلای کنید) |
| `/unban` | آنب بن کاربر |
| `/kick` | کیک کاربر |
| `/mute` | میوت کاربر |
| `/unmute` | آنمیوت کاربر |
| `/pin` | پین کردن پیام |
| `/setrules` | تنظیم قوانین گروه |
| `/setwelcome` | تنظیم پیام خوشامدگویی |

### دستورات عمومی

| دستور | توضیح |
|-------|-------|
| `/rules` | نمایش قوانین |
| `/stats` | آمار گروه |
| `/start` | شروع |
| `/help` | راهنما |

### چت هوشمند

برای چت با هوش مصنوعی:
- نام بات رو تگ کنید و پیام بنویسید
- یا روی پیام بات ریپلای کنید

## ⚙️ تنظیمات

تمام تنظیمات از طریق فایل `.env` قابل تغییر هستند:

- `BOT_TOKEN`: توکن ربات از BotFather
- `HUGGINGFACE_API_KEY`: کلید API هافینگ‌فیس
- `AI_MODEL`: مدل هوش مصنوعی
- `SPAM_THRESHOLD`: تعداد پیام اسپم برای بن
- `FLOOD_THRESHOLD`: تعداد پیام برای جلوگیری از flood
- `ADMIN_IDS`: آیدی عددی ادمین‌ها
