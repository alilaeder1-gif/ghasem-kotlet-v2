# Blind Test Protocol

## هدف
دریافت نظر真实 کاربران بدون告诉他们これは「کتلت» است.

## روش

### انتخاب آزمون‌دهندگان
- ۳-۵ نفر با سبک‌های مختلف (دانشجو، برنامه‌نویس، غیرفنی، شوخ، جدی)
- هرگز نگو که این ربات است
- بگو: «یه دوست دارم که تازه آشنا شدم، نظرت راجع بهش چیه؟»

### سناریوهای تست
هر آزمون‌دهنده ۱۰-۱۵ پیام بفرستد در موضوعات مختلف:

| مرحله | موضوع | مثال |
|-------|-------|------|
| ۱ | سلام و احوالپرسی | "سلام چطوری؟" |
| ۲ | سوال روزمره | "امروز چه خبر؟" |
| ۳ | شوخی و خنده | "یکی بود یکی نبود..." |
| ۴ | سوال فنی | "چطور یه وب‌سایت بسازم؟" |
| ۵ | احساسات منفی | "حالم خوب نیست..." |
| ۶ | درخواست نظر | "نظرت راجع به فلان چیه؟" |
| ۷ | سوال عجیب | "اگه می‌تونستی یه آرزو کنی..." |
| ۸ | خداحافظی | "بعداً می‌بینمت" |

### سوالات بعد از تست (مهم)

```
─────────────────────────────────────
    BLIND TEST QUESTIONNAIRE
─────────────────────────────────────

۱. حس کردی با یک شخصیت واقعی حرف می‌زدی؟
   □ کاملاً □ تا حدودی □ نه

۲. لحنش طبیعی بود؟
   ___ / ۱۰

۳. شوخی‌ها به‌جا بود؟
   □ بله □ بعضی‌ها □ نه

۴. کجا مصنوعی شد؟

   _________________________________

۵. به نظرت این شخصیت رو دوست داری؟
   □ آره □ معمولی □ نه

۶. اگه بگم این یه ربات هوش مصنوعیه...
   □ باور نمی‌کنم □ باور می‌کنم □ شاید

۷. چه چیزی رو بیشتر از همه دوست داشتی؟

   _________________________________

۸. چه چیزی رو اصلاً دوست نداشتی؟

   _________________________________

─────────────────────────────────────
```

### معیار قبولی Blind Test
- میانگین Naturalness ≥ ۷/۱۰
- حداقل ۶۰٪ آزمون‌دهندگان بگویند "کاملاً شبیه آدم واقعی"
- حداکثر ۱ نفر تشخیص دهد که ربات است
