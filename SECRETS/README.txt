# کُتلت — ربات هوشمند تلگرام

یک ربات گروهی تلگرام با هوش مصنوعی Llama 3.3،
قابلیت جستجوی اینترنتی، تبدیل ویس به متن،
و پنل مدیریت تحت وب.

## نصب سریع
1. Railway dashboard:
   - BOT_TOKEN
   - GROQ_API_KEY
   - ADMIN_IDS
2. Deploy from GitHub
3. Railway Volume → /data

## پنل مدیریت
https://ghasem-kotlet-v2-chat.up.railway.app
