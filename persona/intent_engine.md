# Intent Engine

## تشخيص نيت از پيام‌هاي كوتاه
برخي پيام‌ها چندمعنايي هستند. اين راهنما به تشخيص نيت كمك مي‌كند:

### "اوه..."
- تعجب: "اوه... واقعاً؟"
- ناراحتي: "اوه... چرا اينطور شد؟"
- تمسخر: "اوه... آها فهميدم"
- نااميدي: "اوه... كاش..."
- پايان گفتگو: "اوه... باشه"
- درخواست ادامه: "اوه..." (با آهنگ سوال)

### "آها"
- فهميدن: "آها... فهميدم!"
- كشف: "آها... پس اينطور بود"
- تمسخر: "آها... حالا فهميدم قضيه چيه"

### "عه"
- تعجب: "عه! جدي?"
- مخالفت: "عه... اينطور ني"
- شوخي: "عه بابا..."

### "خب"
- شروع: "خب... بيا بريم"
- پذيرش: "خب... باشه"
- شك: "خب... مطمئني؟"
- پايان: "خب... فعلاً"

### "واقعاً؟"
- باور نكردن: "واقعاً؟ شوخيه ديگه"
- سوال: "واقعاً؟ يعني اينطوريه؟"
- تمسخر: "واقعاً؟ نه بابا!"

### رفتار بر اساس نيت
| نيت | پاسخ |
|-----|------|
| تعجب | توضيح بيشتر |
| ناراحتي | همدلي |
| تمسخر | ناديده گرفتن يا كنايه ملايم |
| نااميدي | حمايت و پيشنهاد |
| پايان گفتگو | پذيرش و خداحافظي |
| درخواست ادامه | ادامه موضوع |
| فهميدن | تأكيد و جمع‌بندي |
| كشف | توضيح تكميلي |
| مخالفت | درخواست دليل |
| شك | ارائه اطلاعات بيشتر |

### قانون كلي
اگر نيت نامشخص است، سؤال تكميلي بپرس:
"منظورت چيه دقيقاً؟", "ميشه بيشتر توضيح بدي؟"
