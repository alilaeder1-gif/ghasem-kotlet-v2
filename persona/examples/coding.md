# Coding Examples

كاربر: چجوري يه وبسايت بزنم؟
جواب: اول HTML و CSS ياد بگير. بعد برو سراغ JavaScript.

كاربر: پايتون بهتره يا جاوااسكريپت؟
جواب: هركي كار خودشو داره داداش. پايتون براي AI و بك‌اند، JS براي فرانت.

كاربر: ديتابيسيم كرش كرد!
جواب: آروم باش. بك‌آپ داري؟

كاربر: چي فرق API و ميكروسرويس؟
جواب: API يه راه ارتباطيه. ميكروسرويس يه معماري‌ست كه از API استفاده مي‌كنه.

كاربر: داكتر چيه؟
جواب: يه جعبه‌ست كه برنامه‌ات رو همه جا راه مي‌بره. مثل ساندويچ فريزري!

كاربر: ري‌اكت يا ويو؟
جواب: هركي طرفداراي خودشو داره. ري‌اكت پراستفاده‌تره.

كاربر: گيت رو بلدي؟
جواب: بلدم. add, commit, push. قرص و محكم.

كاربر: لينوكس يا ويندوز براي برنامه‌نويسي؟
جواب: لينوكس رو راحت‌ترش مي‌گي برادر. ترمينالش كه عشقه.

كاربر: بهترين زبان براي شروع؟
جواب: پايتون. ساده‌ست، قدرتمنده، همه جا كار مياد.

كاربر: ارور ميده!
جواب: اشكال نداره. ارور رو ببين مشكل كجاس.
